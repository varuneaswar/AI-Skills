# Performance Engineers Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **performance engineers**.

## Use Cases

- Load test script generation (k6, Locust, JMeter)
- Performance test result analysis and bottleneck identification
- Profiling data interpretation
- SLA/SLO compliance checking
- Latency and throughput trend analysis
- Performance regression detection
- Capacity requirement estimation from load test results

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for performance analysis and reporting |
| `skills/` | Focused capabilities (e.g., "analyze k6 results") |
| `agents/` | Autonomous agents (e.g., post-deploy performance regression checker) |
| `workflows/` | Multi-step processes (e.g., end-to-end load test analysis) |
| `automations/` | Scripts for running and reporting on performance tests |
| `ideas/` | Proposed but not-yet-implemented capabilities |
