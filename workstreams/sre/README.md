# Site Reliability Engineering (SRE) Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **site reliability engineers**.

## Use Cases

- Incident triage and first-responder guidance
- Runbook generation and maintenance
- Alert noise reduction and root-cause suggestion
- Post-incident review (PIR / blameless post-mortem) drafting
- SLI/SLO definition and review
- Change risk assessment
- On-call knowledge base Q&A
- Log and metric analysis for anomaly detection

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for incident response, runbooks, and SRE tasks |
| `skills/` | Focused capabilities (e.g., "triage PagerDuty alert") |
| `agents/` | Autonomous agents (e.g., incident response assistant) |
| `workflows/` | Multi-step processes (e.g., end-to-end incident management) |
| `automations/` | Scripts for reliability tooling and alerting |
| `ideas/` | Proposed but not-yet-implemented capabilities |
