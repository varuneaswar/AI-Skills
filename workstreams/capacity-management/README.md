# Capacity Management Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **capacity management engineers**.

## Use Cases

- Demand forecasting and trend analysis from metrics data
- Resource sizing recommendations (CPU, memory, storage)
- Utilization analysis and headroom calculation
- Capacity planning report generation
- Identifying underutilized or over-provisioned resources
- Cost optimization recommendations

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for capacity analysis and reporting |
| `skills/` | Focused capabilities (e.g., "analyze utilization trends") |
| `agents/` | Autonomous agents (e.g., weekly capacity summary bot) |
| `workflows/` | Multi-step processes (e.g., quarterly capacity planning) |
| `automations/` | Scripts for data collection and report generation |
| `ideas/` | Proposed but not-yet-implemented capabilities |
