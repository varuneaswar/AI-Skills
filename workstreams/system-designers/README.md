# System Designers Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **system designers**.

## Use Cases

- System design document generation and review
- Trade-off analysis (e.g., consistency vs. availability)
- Sequence diagram generation from prose descriptions
- API contract design and review
- Data model design assistance
- Technology selection rationale
- Non-functional requirement elicitation
- Interview-style system design practice and feedback

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for system design tasks |
| `skills/` | Focused capabilities (e.g., "generate sequence diagram") |
| `agents/` | Autonomous agents (e.g., system design reviewer) |
| `workflows/` | Multi-step processes (e.g., system design workshop facilitation) |
| `automations/` | Scripts for design tooling integration |
| `ideas/` | Proposed but not-yet-implemented capabilities |
