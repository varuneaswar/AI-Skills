# Architects Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **solution architects and enterprise architects**.

## Use Cases

- Architecture decision record (ADR) generation and review
- Trade-off analysis (cost vs. performance, scalability vs. simplicity)
- Diagram-as-code generation (C4, sequence diagrams, ER diagrams)
- Technical proposal drafting and review
- Identifying non-functional requirement gaps
- Summarizing technology options and generating comparison matrices

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for architectural tasks (ADRs, RFCs, design reviews) |
| `skills/` | Focused capabilities (e.g., "generate C4 context diagram") |
| `agents/` | Autonomous agents (e.g., architecture review bot) |
| `workflows/` | Multi-step processes (e.g., end-to-end ADR creation) |
| `automations/` | Scripts that automate architecture tooling |
| `ideas/` | Proposed but not-yet-implemented capabilities |
