# Developers Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **software developers**.

## Use Cases

- Code generation, completion, and refactoring
- Pull request summarization and review assistance
- Documentation generation (README, API docs, inline comments)
- Commit message drafting
- Code explanation for onboarding or knowledge transfer
- Test case generation from source code or requirements
- Debugging assistance and root-cause analysis
- Dependency update impact analysis

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for coding, reviewing, and documenting code |
| `skills/` | Focused capabilities (e.g., "explain this function") |
| `agents/` | Autonomous agents (e.g., PR review bot) |
| `workflows/` | Multi-step processes (e.g., feature-branch PR workflow) |
| `automations/` | Scripts that automate development tooling |
| `ideas/` | Proposed but not-yet-implemented capabilities |
