# Testers Work Stream

This folder contains AI skills, prompts, agents, workflows, and automations designed for **QA engineers and testers**.

## Use Cases

- Test case generation from requirements, user stories, or source code
- Boundary value and equivalence partition analysis
- Exploratory test charters
- Test plan generation
- Bug report drafting and triaging
- Regression suite gap analysis
- Accessibility testing guidance
- API test script generation (Postman, REST-assured, pytest)
- Performance test scenario creation

## Contents

| Folder | What Belongs Here |
|---|---|
| `prompts/` | Prompts for testing tasks |
| `skills/` | Focused capabilities (e.g., "generate test cases from user story") |
| `agents/` | Autonomous agents (e.g., test coverage analyst) |
| `workflows/` | Multi-step processes (e.g., requirements-to-test-suite workflow) |
| `automations/` | Scripts for test tooling automation |
| `ideas/` | Proposed but not-yet-implemented capabilities |
